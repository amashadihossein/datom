# Download File from Storage

Download File from Storage

## Usage

``` r
.datom_storage_download(conn, key, local_path)
```

## Arguments

- conn:

  A `datom_conn` object.

- key:

  Relative storage key (after `prefix/datom/`).

- local_path:

  Local file path (destination).

## Value

Invisible `TRUE` on success.
