# Upload File to Storage

Upload File to Storage

## Usage

``` r
.datom_storage_upload(conn, local_path, key)
```

## Arguments

- conn:

  A `datom_conn` object.

- local_path:

  Local file path to upload.

- key:

  Relative storage key (after `prefix/datom/`).

## Value

Invisible `TRUE` on success.
